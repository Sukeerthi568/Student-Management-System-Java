# Student Management System (Java)

## Project Description
The Student Management System is a simple Java-based application that allows users to manage student records efficiently. The system provides basic operations such as adding, viewing, searching, and deleting student information.

## Features
- Add new student details
- View all student records
- Search student by ID
- Delete student record
- Menu-driven console interface

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner Class
